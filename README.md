# Real-Time Mobile App with Notifications

## Project Documentation

### Architecture
This project is designed with a clean architecture for mobile applications using Flutter. It follows the principles of separation of concerns, making it easy to maintain and scale. The architecture consists of the following layers:

- **Presentation Layer**: Contains UI components and handles user interactions.
- **Domain Layer**: Contains business logic and use cases.
- **Data Layer**: Manages data sources, including remote APIs and local databases.

### Setup Instructions
1. **Install Flutter**: Make sure to have Flutter installed on your machine. You can follow the installation guide at [Flutter Install](https://flutter.dev/docs/get-started/install).
2. **Clone the Repository**:
   ```bash
   git clone https://github.com/M-K-Chaitanya/Real-Time-Mobile-App-with-Notifications.git
   cd Real-Time-Mobile-App-with-Notifications
   ```
3. **Dependencies**: Run the following command to install dependencies:
   ```bash
   flutter pub get
   ```
4. **Run the App**:
   Make sure you have an emulator running or a device connected, then execute:
   ```bash
   flutter run
   ```

### Features
- Real-time notifications using Firebase Cloud Messaging.
- Offline support using local storage.
- Beautiful and responsive UI using Flutter widgets.
- User authentication options with Firebase.
- Push notifications management.
